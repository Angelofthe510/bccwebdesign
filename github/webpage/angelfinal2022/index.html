A Simple Final Project

        Fall 2022

This semester, I took MMDI 4 Intro to Photoshop, MMVI 20A Intro to Video Editing, and MMWW 1A Intro to Web Design. I also signed up for the free MMDI 515A Intro to C4D class.

I've taken Motion Graphics, Game Design, Intro to MultiMedia, Script Writing, and Intro to Video Production at BCC. With all the classes, we have blended assets from Photoshop, Illustrator, After Effects, original footage and audio, and stock video and audio into various projects.

Once I started getting going in my classes, I became excited about all the different techniques that I would learn. While going through the first few weeks of classes, I began to think about how I could incorporate projects between classes. My first collaboration was with the Sugar Poster. Using the Team Project from C4D, I was able to create a poster from the final shot of our C4D video.

The next project sorta fell out of the sky. A Split Screen for Intro to Video soon became a short video to show the differences between how two textures in C4D render during animation. The Final Project for Intro to Photoshop seemed like an easy thing to pull off. All I needed was a subject.

One day diving down a rabbit hole on the internet, I came across a story about how Jefferson Starship had regrouped and recorded new material, but of course the first footage I saw was the new JS performing White Rabbit. I decided that my final for Video Editing would be a mash-up of the old WR playig along with the new WR.

The Final Project developed into a crossover with C4D, Photoshop, Illustrator, Premier, and After Effects to create promotional material for a music group.

Components of the Video

C4D: The Pills

For the video, I created two backdrops for the pills, one motif is just the Red and Blue pills, the other motif is the Yellow, Lt Blue, Red and Blue pills.

First, I set up the C4D file to match the file size of output. Next, I created the two pills, the textures, camera, and lights then began experimenting with the settings to get a look I liked.

Whats cool about this, is that all the elements are editable so I was able to twist, turn and position the pill clusters where I wanted. Each screensize required a slightly different placement of elements, so I created three files for eachy size.

Once the file was created, I exported each frame as a seperate Photoshop document. I then was able to edit the opacity of the background.

I used multiple files for the pills. One file was just for the animation. It was set to the screensize of the video then rendered to a video that was then imported into the Premier project.

I also had made the "My Turntable" movie on my own during the summer break. Using the techniques learned in the MMDI 15A class I made the turntalbe movie just for fun. I thought I would try and use it for this project, but it was cut during the review process. No big deal though, I like the end result much better.

MMDI 15A with Michel Bobhot is the class to teach you all these great skills to make all kinds of objects which can be used in Photoshop.
